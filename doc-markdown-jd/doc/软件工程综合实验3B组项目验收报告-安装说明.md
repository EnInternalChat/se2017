##安装说明
# 
#

### 一、自动部署服务器要求
1.服务器/usr/share目录下没有名为Backend和web-server的目录
2.服务器当前操作用户为root用户
3.系统已按正常路径安装好了docker并正常启动了docker服务
4.系统已按正常路径安装好了git
5.主机的80、8080、8888、443等端口并未被其他服务占用
### 二、部署流程
1.切换到服务器/usr/share目录下
2.启动autoInitService.sh脚本
3.等待部署完成迎
